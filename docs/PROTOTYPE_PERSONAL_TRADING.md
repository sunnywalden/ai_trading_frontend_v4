## 0. 原型说明

- 本原型已扩展为**高保真视觉规范**，包含布局、颜色、字号与组件交互细节。
- 页面结构与组件名称与实现文件一致（`src/views` / `src/components`）。

## 1. 页面流（全局导航）

```text
[登录后默认]
  -> / (Dashboard)
  -> /behavior
  -> /positions
  -> /opportunities (Execution List)
  -> /macro
  -> /monitoring (System Health)
```

┌─────────────────────────────────────────────┐
│ Header：行为评分 + 窗口天数                 │

## 9. 交互流程（关键路径）

```text
用户进入首页
  -> 查看风险水位
  -> 若风险超限：跳转持仓评估
  -> 发现计划偏离：跳转行为评分
  -> 在执行列表中筛选并加入计划
  -> 盘后复盘入口
```

## 10. 高保真视觉规范

### 10.1 颜色 Token

| Token | 颜色 | 用途 |
| --- | --- | --- |
| `--bg-primary` | `#020617` | 主背景 |
| `--bg-secondary` | `#0f172a` | 次级背景 |
| `--bg-card` | `rgba(15, 23, 42, 0.85)` | 卡片背景 |
| `--primary` | `#38bdf8` | 主按钮/链接 |
| `--success` | `#22c55e` | 正常/上涨 |
| `--warning` | `#f59e0b` | 注意/偏离 |
| `--danger` | `#ef4444` | 风险/错误 |
| `--text-primary` | `#e5e7eb` | 主文本 |
| `--text-secondary` | `#9ca3af` | 次文本 |

### 10.2 字体与层级

| 层级 | 字号 | 字重 | 用途 |
| --- | --- | --- | --- |
| H1 | 22px | 600 | 页面标题 |
| H2 | 18px | 600 | 区块标题 |
| Body | 14–16px | 400 | 正文 |
| Caption | 12px | 500 | 标签/辅助 |

### 10.3 栅格与间距

- 页面最大宽度：1200px
- 左右边距：24px
- 栅格：12 列
- 列间距：16px
- 卡片内边距：16px
- 区块间距：16–24px

## 11. 组件高保真规格

### 11.1 SummaryBar

- 高度：72px
- 背景：`--bg-card`
- 文本：左对齐，分组间距 24px

### 11.2 PositionScoreCard

- 宽度：360~420px（自适应）
- 高度：220px（不含趋势折叠）
- 边框：`rgba(56, 189, 248, 0.2)`
- Hover：上移 1px + 边框增强

### 11.3 TrendSnapshotCard

- 高度：200px（折叠后）
- 进度条高度：8px
- 空态按钮：主色描边 + 透明底

### 11.4 ExecutionItemCard

- 高度：180px
- 标签：12px，圆角 6px
- 匹配理由：最多 2 行省略

### 11.5 SystemHealthCard

- 高度：120px
- 状态图标：24px
- 重试按钮：44px 高

## 12. 交互与状态样式

### 12.1 风险级别样式

| 等级 | 背景 | 边框 | 文案 |
| --- | --- | --- | --- |
| LOW | `rgba(34,197,94,0.08)` | `#22c55e` | 绿色 |
| MEDIUM | `rgba(245,158,11,0.08)` | `#f59e0b` | 橙色 |
| HIGH | `rgba(239,68,68,0.08)` | `#ef4444` | 红色 |

### 12.2 按钮状态

- 默认：主色实心
- Hover：亮度 +10%
- Disabled：透明度 40%，禁用指针

### 12.3 空态

- 图标：48px
- 文案：`--text-secondary`
- 按钮：主色描边
